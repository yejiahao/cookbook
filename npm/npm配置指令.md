### 配置淘宝镜像源
```
$ npm install -g cnpm --registry=https://registry.npm.taobao.org
``` 

### 安装全局工具
```
$ cnpm install gulp-cli -g
```

### 查看版本
```
$ node -v
v10.16.0

$ npm -v
6.9.0
```

### 安装前端类库
```
$ npm install jquery@1.12.4
$ npm install angular@latest --save
```

### 查看远程前端类库
```
$ npm view vue version(s)
2.6.10
```
